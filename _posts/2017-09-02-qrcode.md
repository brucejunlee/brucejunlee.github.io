本文使用python第三方库，简单编写一个二维码解析和生成的程序。

## 第三方库
+ qrtools
+ pypng
+ pyqrcode
+ zbar

## 示例程序
下面程序在python3.5环境下测试通过

```python
import pytools
import pypng
import pyqrcode

qr = pyqrcode.create("brucejunlee.site")
qr.png("myQRcode.png", scale=6)

qr = qrtools.QR()
qr.decode("myQRcode.png")
print qr.data
```
